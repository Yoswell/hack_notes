### Cryptography

  * [Dencode](https://dencode.com): Encode/decode various formats (Base64, Hex, URL, etc.).
  * [Boxentriq Cipher Identifier](https://www.boxentriq.com/code-breaking/cipher-identifier): Automatically identifies cipher types from encrypted text.
  * [CrackStation](https://crackstation.net): Online hash cracker for MD5, SHA1, etc. (Use wordlists for better results.)
  * [dcode Multi-tap ABC Cipher](https://www.dcode.fr/multitap-abc-cipher): Decodes T9-style mobile keypad ciphers.
  * [dCode Cipher Identifier](https://www.dcode.fr/cipher-identifier): Broad cipher identification and decoding.
  * [AnyCript](https://anycript.com/crypto): Encryption/decryption tools for symmetric/asymmetric ciphers.
  * [FactorDB](https://factordb.com): Database for integer factorization, useful for RSA challenges.
  * [UTF-8 Character Table](https://www.utf8-chartable.de/unicode-utf8-table.pl): Lookup Unicode characters and encodings.
