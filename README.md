### CTF Notes

Personal repository of writeups, notes, and techniques used in **CTF (Capture The Flag) challenges**. This repository serves as a comprehensive knowledge base for cybersecurity enthusiasts, covering a wide range of topics from various platforms and categories including forensics, web exploitation, cryptography, reverse engineering, OSINT (Open Source Intelligence), binary exploitation (pwn), steganography, and miscellaneous challenges.

The goal is to document practical techniques, tools, and methodologies encountered during CTF competitions, providing quick references and in-depth explanations to aid in solving similar challenges. Each section includes code snippets, examples, and best practices to enhance learning and application in real-world scenarios.

-----

### How to Use This Repository

* Navigate to specific categories for targeted information.
* Use the cheatsheets for quick command references.
* Refer to writeups for step-by-step solutions to challenges.
* Contribute by adding new notes or improving existing ones.

For additional resources, check the [cheatsheets](/cheatsheets/) directory for quick references on tools and techniques.
